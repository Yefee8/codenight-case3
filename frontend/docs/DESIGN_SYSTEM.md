# Paycell-inspired design system
This is an adaptation, not a replica. The reference is the current [Paycell website](https://paycell.com.tr/), inspected on 22 July 2026.

## Verified reference tokens
| Token | Reference | FraudCell use |
|---|---|---|
| Deep blue | `#034EA2` | Brand text and navigation |
| Blue | `#2F7DFA` | Primary gradient start/middle |
| Aqua | `#22CDDB` | Primary gradient end and focus |
| Pale surface | `#F7FAFC` | App background |
| Ink | `#1C1C27` | Primary light-theme text |
| Muted ink | `#8E8E93` | Secondary copy |
| Campton | Display family | Titles and product identity |
| Roboto | UI family | Dense tables, forms and body copy |

Paycell’s published CSS uses a blue-to-aqua gradient and rounded calls to action. FraudCell keeps those cues but uses tighter spacing, restrained shadows and semantic fraud colors for a modern operations product.

Dark mode maps the same family to `#091229` and `#0C2037`, both colors present in the reference stylesheet, while retaining accessible foreground contrast.
